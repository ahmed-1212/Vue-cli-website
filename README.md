# Vue CLI Website 

`Construction theme website build with vue cli using`

* Vue CLI
* Vue Router
* Vue x
* Jquery
* SASS
* Bootstrap 4

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
